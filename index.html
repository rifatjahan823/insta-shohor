Local Storage

1.The data of local storage remains permanently in the browser and data are saved even when the browser is closed.

2.Local storage is created not only for one page and but it also does not clear data from one page to another

3. Local storage enables developers to save and retrieve different values.

Session-storage

1.The data of session storage remains temporary in the browser and data are deleted even when the browser is closed.

2.Session storage is created only for one page, it clears data from one page to another

3.Session storage does not enable developers to save and retrieve different values.
